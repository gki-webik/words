Заголовки

Для создания заголовков используются теги:

  • <h1>...</h1>

  • <h2>...</h2>

  • <h3>...</h3>

  • <h4>...</h4>

  • <h5>...</h5>

  • <h6>...</h6>

Шрифт

Для форматирования и выделения шрифта используются теги:


  • <b>...</b> - Жирный

  • <i>...</i> - Курсив

  • <u>...</u> - Подчеркнутый

  • <tt>...</tt> - Равноширный (Использование этого элемента осуждается спецификацией HTML, вместо него применяйте стили (font-family: Courier, monospace;))

  • <em>...</em> - Курсив (Семантический тег)

  • <strong>...</strong> - Жирный (Семантический тег)

  • Жирный (<b>...</b>)
  • Курсив (<i>...</i>)
  • Подчеркнутый (<u>...</u>)
  • Равноширный (<tt>...</tt>)
  • Курсив (Семантический тег) (<em>...</em>)
  • Жирный (Семантический тег) (<strong>...</strong>)

Списки

Для создания списков используются теги:


  • Нумерованный список (<ol><li>Элемент списка</li></ol>)

  • Маркированный список (<ul><li>Элемент списка</li></ul>)

Термины

Для создания каких-либо терминов используются теги:


  • <dl>...</dl> (Контейнер для термина)

  • <dt>...</dt> (Заголовок)

  • <dd>...</dd> (Описание)
